# AutoFlow - Workflow Automation Platform

A modern, responsive landing page for AutoFlow, a workflow automation platform that helps teams streamline their processes and boost productivity.

## Features

- **Responsive Design**: Fully responsive across all devices (mobile, tablet, desktop)
- **Modern UI**: Dark theme with sky blue and emerald gradient accents
- **Smooth Animations**: Hover effects, transitions, and scroll animations
- **Component Architecture**: Modular, reusable React components
- **Type Safe**: Built with TypeScript and Next.js 16
- **Performance Optimized**: Fast loading times with Next.js App Router

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4
- **UI Components**: shadcn/ui
- **Icons**: Lucide React
- **Language**: TypeScript
- **Deployment Ready**: Vercel-optimized

## Design Highlights

- **Color Palette**: Dark background (#0a0a0a) with sky-500 and emerald-400 accents
- **Typography**: Inter font family for clean, modern text
- **Spacing**: Consistent padding and margins using Tailwind's spacing scale
- **Accessibility**: Semantic HTML, proper ARIA labels, and keyboard navigation
- **Glass Morphism**: Subtle backdrop blur effects on cards

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)

## Deployment

This project is optimized for deployment on Vercel:

1. Push to GitHub
2. Import project in Vercel
3. Deploy with zero configuration
